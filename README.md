# Vinci Oracle

## Setup

Requirements:

- npm

```
$ git clone git@github.com:VinciProtocol/vinci-oracle.git
$ cd vinci-oracle
$ npm i        # Install dependencies
```

Create a copy of the file `env.tmpl`, and name it `.env`.


## Compile

```
npx hardhat compile
```

## Deployment

```
npx hardhat run scripts/script_name.js --network network
```
